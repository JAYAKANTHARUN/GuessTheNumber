## Run Locally
- Open cmd in New Folder 
- Clone the project

```bash
  git clone https://https://github.com/JAYAKANTHARUN/GuessTheNumber
```

- Go to the project directory

```bash
  cd GuessTheNumber
```

- Compile the game

```bash
  javac guessthenumber.java
```

- Start the game

```bash
  java guessthenumber
```
